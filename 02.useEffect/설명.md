## useEffect

---

- 어떤 컴포넌트가 `mount(화면에 첫 렌더링)`, `update(다시 렌더링)`, `unmount(화면에서 사라질때)` 특정 작업을 처리할 코드를 사용하고 싶다면 `useEffect`를 사용하면된다.
- 인자로 콜백함수를 가지며 원하는 작업을 작성하면 된다.

- 두가지 형태가 있다.
  1. 콜백함수만 받는 경우
     - 렌더링 될 때 마다 실행

  2. 첫번째 인자로 콜백함수, 두번째 인자로 `dependency array`를 가진다.
    - 매번 렌더링될때마다 실행되는 것이 아니라 화면에 첫 렌더링 될때와 배열 내의 value값이 바뀔때 실행됨
    - 만일 빈배열만 전달해주면 컴포넌트가 맨처음 렌더링 될 때만 실행됨

--- 

### Clean up
- 정리 작업
- 구독해놓았다가 해제해주는 작업 
- 정리작업을 해주려면 useEffect내에서 함수를 return해주면됨
- 클린업 함수는 useEffect의 return 값으로 정의하는데, 컴포넌트가 삭제될 경우 함수가 실행된다.

### 형태

```js
useEffect(() => {
    (이펙트 함수)
    return {
        (클린업 함수)
    };
}, [의존값]);
```
